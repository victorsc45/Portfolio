Runs in the browser and feature dynamically updated HTML and CSS

Weather dashboard holds multiple cities search in local storage persistently

Developed using bootstrap, jQuery and dynamically updating the html and css. Accepts user input of any city and upon searching that city name will retrieve weather information Using server-side technologies for API from openweather.org created a current weather and forecast of weather for 5 days. User can look-up multiple cities and click back and forth to see the weather conditions.
