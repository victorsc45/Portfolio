User can search the OMDB API for Movies

The queries are filtered by content providers HULU, Amazon Prime, GooglePlay, Itunes, and Netflix

The second API is consumed for related movies or series based on top billing actor

Stream bingers was developed using Bulma.io, jQuery, Javascript, HTML, and CSS. The app consumes external API content from two sources using ajax queries. The APIs where limiting; the cast was set in an array and filtered by top billing to get related content.
